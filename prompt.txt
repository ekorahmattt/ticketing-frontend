Buatkan tampilan **Admin Dashboard untuk Sistem Ticketing IT Support** menggunakan **React JS (Vite)** dengan desain modern, responsif, modular, dan scalable.

Gunakan layout dashboard profesional dengan:

* Sidebar navigation di kiri
* Topbar di atas
* Main content area
* Dukungan Dark Mode dan Light Mode
* UI bersih dan mudah digunakan oleh admin IT

Gunakan dummy data dan komponen reusable.

---

# Struktur Folder yang Wajib Digunakan

Pisahkan halaman dan komponen agar modular dan mudah dikembangkan.

Gunakan struktur:

src/

* pages/ → hanya berisi halaman utama
* components/ → berisi semua komponen reusable
* components/layout/ → komponen layout global
* components/ui/ → komponen UI umum
* components/feature-specific/ → komponen khusus fitur

Jangan menaruh komponen reusable di dalam folder pages.

---

# Struktur Menu Sidebar

1. Dashboard
2. Monitoring Laporan
3. Kelola Perangkat

# 1. Halaman Dashboard

## Statistik Laporan

Stat cards:

* Total laporan
* Laporan hari ini
* Laporan selesai
* Laporan diproses
* Laporan terbuka

Filter:

* Rentang waktu
* Status
* Kategori
* Unit / lokasi (opsional)

Tambahkan grafik distribusi laporan.

---

## Riwayat Laporan (Selain Hari Ini)

Tabel riwayat laporan lampau.

Kolom:

* ID laporan
* Jenis gangguan
* Pelapor / unit
* Status
* Prioritas
* Tanggal laporan
* Tanggal selesai

# 2. Halaman Monitoring Laporan Gangguan

Command center real-time.

## Statistik Hari Ini

* Diproses

## View Switch: Daftar ↔ Denah

Tambahkan komponen ViewSwitcher.

* List View
* Denah View

---

## List View — Laporan Hari Ini

Tabel real-time:

* Waktu laporan

* Jenis gangguan
* Status
* Prioritas

---

## Denah View — Lokasi Laporan

Denah interaktif:
